## Setup
1. **Clone the repo**
```
$ git clone git@gitlab.com:educastle/sample/comicsapp.git
$ cd comicsapp
$ flutter packages get
```
2. **Running:**
```
$ flutter run
$ Tip:if build fail,just build again.
$ flutter version : 1.0.0
```
